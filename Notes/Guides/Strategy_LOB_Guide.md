# LOB 全方位策略指南：從大戶籌碼到微結構訊號

本指南整合了訂單流 (Order Flow)、掛單簿 (Order Book) 與大戶籌碼 (Volume Profile) 的綜合分析邏輯。

---

## 1. 核心指標定義

### A. 訂單流指標 (Flow Indicators) - 攻擊方
這類指標反映市場的**主動攻擊力道**。

1.  **CVD (Cumulative Volume Delta) - 累積成交差**
    *   **來源**: `成交明細 (Ticks)`
    *   **定義**: `主動買成交量 - 主動賣成交量` 的累加值。
    *   **意義**: **真金白銀的戰果**。代表買賣雙方實際「吃掉」了多少對手盤。最直觀的多空力道。

2.  **CumOFI (Cumulative Order Flow Imbalance) - 累積訂單流不平衡**
    *   **來源**: `最佳五檔報價變化 (Bid/Ask Quotes)`
    *   **定義**: `Sum(5檔委買量變化) - Sum(5檔委賣量變化)`。
    *   **意義**: **全深度的攻擊 + 意圖**。
        *   我們的引擎計算了**所有五檔**的掛單變化，不僅僅是最佳一檔。這意味著即使主力是在底下幾檔默默撤單，也會被 OFI 偵測到。
        *   若有人主動買進 -> Ask 減少 -> OFI 上升。
        *   若有人撤銷賣單 -> Ask 減少 -> OFI 上升 (代表賣壓減輕)。
    *   **特點**: 比 CVD 更敏銳，包含了「撤單讓路」的隱性多頭訊號。

### B. 掛單簿指標 (Book Indicators) - 防守方
這類指標反映市場的**被動防守意圖**。

3.  **CumOBI (Cumulative Order Book Imbalance) - 累積掛單簿不平衡**
    *   **來源**: `最佳報價狀態 (Bid/Ask Quotes)`
    *   **定義**: `(委買量 - 委賣量) / (委買量 + 委賣量)` 的累加值。
    *   **關鍵時機**: **Tick 發生當下的瞬間快照 (Snapshot on Tick)**。
        *   這點非常重要。我們記錄的是「當有人成交時，掛單簿長什麼樣子」。
        *   這代表了成交當下，市場提供的防守深度是多少。
    *   **意義**: **防守深度與意圖**。
        *   OBI 上升：Tick 發生時，下方委買很厚。
        *   OBI 下跌：委賣更厚 (上方蓋頭反壓 / 下方買盤撤退)。
    *   **特點**: 通常作為「背離」指標，觀察主力是在「擋單(吸籌)」還是「抽單(放棄)」。

### C. 大戶籌碼指標 (Lot Size) - 控盤者
這類指標將成交量依據單筆大小分類，觀察誰在主導市場。

4.  **Small Lot (小單 < 5口)**: 散戶指標。通常是反向指標。
5.  **Large Lot (大單 >= 5口)**: 主力/程式單。趨勢的主要推動者。
6.  **Mega Lot (特大單 >= 15口)**: 法人/造市商。關鍵轉折點常伴隨特大單出現。

## 2. 訊號解讀與策略矩陣

我們將市場情境分為四大類：趨勢、吸籌、出貨、虛漲/虛跌。

### A. 趨勢確認 (Trend Confirmation)
當所有指標同向時，趨勢最健康，勝率最高。

*   **強勢多頭 (Strong Bull)**
    *   **價格**: ↗️ 創新高
    *   **OFI / CVD**: ↗️ 同步創新高 (買盤積極進攻)
    *   **OBI**: ↗️ 同步上升 (買盤支撐跟進 / 賣壓撤退)
    *   **策略**: **追價做多 / 拉回加碼**。

*   **強勢空頭 (Strong Bear)**
    *   **價格**: ↘️ 創新低
    *   **OFI / CVD**: ↘️ 同步創新低 (賣盤積極倒貨)
    *   **OBI**: ↘️ 同步下降 (賣壓蓋頭 / 買盤撤退)
    *   **策略**: **順勢放空 / 反彈加碼**。

### B. 吸收與反轉 (Absorption & Reversal)
當價格創新低/新高，但指標卻沒有跟隨 (背離)，代表「動能被牆擋住了」。

*   **底部吸籌 (Bullish Absorption)**
    *   **價格**: ↘️ 創新低 (看似還在跌)
    *   **OFI / CVD**: ↘️ 創新低 (賣盤還在殺)
    *   **OBI**: **↗️ 更高的低點 / 持平** (關鍵！)
    *   **解讀**: 雖然有人在殺，但下方的限價買單(Bids)越掛越厚，把賣壓全吃光了。
    *   **策略**: **等待價格止穩後做多**。
    *   **經典案例**: **12/12 夜盤 (01:00 AM 後)**。
        *   價格持續創新低，且 OFI 疲弱。
        *   但 OBI 卻從 01:00 開始逆勢墊高，形成完美的「底背離」。
        *   這暗示主力正在低檔吸籌，是後市反彈的強力訊號。

*   **頂部出貨 (Bearish Absorption)**
    *   **價格**: ↗️ 創新高
    *   **OFI / CVD**: ↗️ 創新高 (買盤還在追)
    *   **OBI**: **↘️ 更低的高點 / 持平**
    *   **解讀**: 雖然有人在買，但上方的限價賣單(Asks)源源不絕，漲不上去。
    *   **策略**: **等待價格跌破支撐後放空**。

### C. 虛漲/虛跌 (Fakeout / Hollow Move)
這是最危險也最暴利的訊號。價格與**所有指標**都背離。

*   **虛漲 (Hollow Rise / Vacuum Rise)**
    *   **價格**: ↗️ 緩步或垂直創高 (Drifting/Vacuum Up)
    *   **OFI / CVD**: **↘️ 往下** (根本沒有主動買盤，甚至在偷賣)
    *   **OBI**: **↘️ 急劇往下** (這點最弔詭)
        *   **微結構原理**: 為什麼上方賣單撤了 OBI 還會跌？
        *   因為**下方買單撤得更快**！或者根本沒人掛買單。
        *   這代表市場進入「真空狀態 (Liquidity Vacuum)」。上方沒壓力，下方沒支撐。價格像氣球一樣飄上去。
    *   **解讀**: 這種上漲源於流動性枯竭。極度脆弱，一旦回檔就是自由落體。
    *   **策略**: **極度危險。切勿追高。觀察 OBI 是否在低檔躺平，準備重倉放空。** (如 12/10, 12/11 夜盤案例)

*   **虛跌 (Hollow Drop / Vacuum Drop)**
    *   **價格**: ↘️ 緩步創低
    *   **OFI / CVD**: **↗️ 往上** (主動盤其實在買)
    *   **OBI**: **↗️ 往上** (下方支撐很強)
    *   **解讀**: 價格下跌只是因為沒人掛買單(真空)，但實際上主力在吃貨。
    *   **策略**: **準備強勢反彈做多**。

### D. 特殊形態解讀 (Special Patterns)

*   **軋空停損盤 (The Stop Run / Short Squeeze)**
    *   **場景**: 此型態常見於夜盤或突破關鍵壓力位時 (如 12/10 04:00 AM)。
    *   **OFI 形狀**: **「Spike 噴出 -> Plateau 高原 -> Drop 下墜」**
    *   **階段解析**:
        1.  **Spike (噴出)**: 價格觸及空單痛點，觸發大量 **Buy Stops (市價買單停損)**。OFI 瞬間暴衝。這是**被動買盤**，非主力進場。
        2.  **Plateau (高原)**: 停損結束後，由於沒有新的主力買盤，也沒有人敢空，OFI 停在高檔不動。價格靠慣性懸浮。
        3.  **Drop (下墜)**: 尾盤或力竭時，當沖客獲利了結，OFI 快速回落。
    *   **解讀**: 這是判斷 **「假突破」** 的鐵證。真突破的 OFI 應該持續墊高，而非形成高原。
    *   **策略**: **Plateau 階段是誘多陷阱，切勿追高。等待 Drop 階段做空。**
